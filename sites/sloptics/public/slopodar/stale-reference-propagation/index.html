Stale Reference Propagation

detected 2026-03-02

trigger

"Clean session Weaver reports "13-layer harness model" and "Lexicon at v0.17" — both were stale. The file was deleted; the version was hallucinated."

what it is

When configuration documents describe a state that no longer exists, every agent that boots from them will hallucinate the described state into reality. Unlike human documentation rot (which degrades through neglect), agentic documentation rot is actively consumed as truth on every boot. It propagates.

what it signals

instead

Every structural change must update every document that references the changed structure. The cost of not doing it is 50 sessions built on a false premise.
